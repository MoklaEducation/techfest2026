The Goblin Snack Boxes

Input format used in test files:
First line: number of test cases T
For each test case: N followed by N box values followed by M

Example:
3
5 5 8 11 12 8 10
3 1 3 5 4
4 20 20 19 2 20

Output format:
Even total = X, Big boxes = Y

Files:
goblin.1.in / goblin.1.out - official sample tests
goblin.2.in / goblin.2.out - edge and medium tests
goblin.3.in / goblin.3.out - 5000 randomized tests
solution.py - Python solution
Solution.java - Java solution
solution.cpp - C++ solution

Points: 15

- 6 lloops
- 3 nested loop
- 1 even
- 2 comparison
- 3 output
